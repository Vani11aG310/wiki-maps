### Routes that we will need for our application

## Map routes

* CREATE:       POST /maps 
* READ:
  * All maps:   GET /maps
  * Single map: GET /maps/:id
UPDATE:         POST /maps/:id
DELETE:         DELETE /maps/:id

## User routes

* CREATE:   POST /users
* READ:     GET /users/:id
* UPDATE:   POST /users/:id
* DELETE? : DELETE /users/:id
